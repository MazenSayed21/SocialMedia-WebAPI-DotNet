    %% Core Entities

    class User {

        +int id

        +String name

        +String password

        +String email

        +String bio

        +String profile_photo_url

    }



    class Post {

        +int id

        +int creator_id

        +String url_photo

        +String caption

        +String content

    }



    class Comment {

        +int id

        +int creator_id

        +int post_id

        +DateTime timestamp

        +int parent_id

    }



    class Reaction {

        +String react

        +int creator_id

        +int post_id

    }



    class Notification {

        +int id

        +int receiverId

        +String message

        +String type

        +DateTime timestamp

        +bool isRead

    }

   User "1" -- "*" Post : creates

    %% Relationships (The "Who does What")


    IFriendshipService <|.. FriendshipService : implements

    FriendshipService --> IFriendshipRepo : uses/depends on

    FriendshipService --> IUserRepo : validates users



    IPostService <|.. PostService : implements

    PostService --> IPostRepo : Dependency Injection

    PostService --> IUserRepo : Dependency Injection


    IUserRepo <|.. SqlUserRepo : implements

    IPostRepo <|.. SqlPostRepo : implements

    ICommentRepo <|.. SqlCommentRepo : implements

    IReactionRepo <|.. SqlReactionRepo : implements



    NotificationService --> IUserRepo : uses

    INotificationService <|.. NotificationService : implements

    NotificationService --> INotificationRepo : uses

    NotificationService --> Notification : manages


    ICommentService <|.. CommentService : implements
  
    CommentService --> ICommentRepo : uses

    CommentService --> IPostRepo : validates post existence

    CommentService --> INotificationService : notifies post owner


    IReactionService <|.. ReactionService : implements

    ReactionService --> IReactionRepo : uses

    ReactionService --> IPostRepo : checks post existence

    ReactionService --> INotificationService : alerts post owner
